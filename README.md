# HIGGINS MODS - Modded APK Download Site

A single-page, dark-themed website for hosting and downloading modded APK files. Only administrators can upload files, while users can browse and download available APKs.

## Features

- **Dark Theme with Luminous Purple Accents**: Modern dark interface with glowing purple elements
- **Admin Authentication**: Secure login system for administrators
- **APK Management**: Upload, display, and download APK files
- **Admin Controls**:
  - Upload new APK files
  - Change admin password
  - Manage APK listings
- **User Features**:
  - Browse available modded APKs
  - Download APK files
  - Contact admin via WhatsApp
- **Responsive Design**: Works on desktop and mobile devices
